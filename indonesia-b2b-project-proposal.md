## 一、项目背景与目标

### 1.1 市场机会

印尼 B2B 二手电脑市场规模庞大，企业 IT 设备采购需求旺盛。通过对标印尼市场成功玩家 REF GROUP INDONESIA（2022 年进入印尼，已完成超过 11,600 个项目），验证了"大品牌翻新电脑"在印尼 B2B 市场的真实需求。

**核心机会**：
- 印尼企业对 IT 设备成本敏感，翻新设备可节省 40-60% 采购成本
- 大品牌（Dell、HP、Lenovo）翻新设备在印尼有成熟需求
- B2B 采购决策周期长（1-3 个月），信任是关键，需要专业的独立站和运营体系

### 1.2 项目目标

为客户打造一套**完整的 B2B 获客系统**，分两阶段交付：

**第一阶段（第 1 个月）**：
1. **独立站**：面向印尼 B 端客户的专业供应商官网（Next.js 完全自研，视觉独特）
2. **运营策略**：经过市场验证的 B2B 获客方法论
3. **运营系统**：整合第三方工具的自动化运营平台（Golang 高性能后端，统一管理后台）

**第二阶段（第 2 个月）**：
4. **AI 投流支持**：基于数据反馈的智能投流优化

**最终目标**：帮助客户在 6 个月内建立稳定的 B 端客户获取渠道，月 GMV 达到 $20,000+。

### 3.3 运营系统（Golang 后端）

#### API 设计

**内容管理 API**：
```
GET    /api/admin/products          # 获取产品列表
POST   /api/admin/products          # 创建产品
GET    /api/admin/products/:id      # 获取单个产品
PUT    /api/admin/products/:id      # 更新产品
DELETE /api/admin/products/:id      # 删除产品

GET    /api/admin/cases             # 获取案例列表
POST   /api/admin/cases             # 创建案例
# ...

GET    /api/admin/articles          # 获取文章列表
POST   /api/admin/articles          # 创建文章
# ...

POST   /api/admin/media/upload      # 上传媒体文件
GET    /api/admin/media             # 获取媒体列表
```

**运营管理 API**：
```
GET    /api/admin/customers         # 获取客户列表
POST   /api/admin/customers         # 创建客户
GET    /api/admin/customers/:id     # 获取客户详情
PUT    /api/admin/customers/:id     # 更新客户

GET    /api/admin/inquiries         # 获取询盘列表
POST   /api/admin/inquiries         # 创建询盘
PUT    /api/admin/inquiries/:id     # 更新询盘状态

GET    /api/admin/orders            # 获取订单列表
POST   /api/admin/orders            # 创建订单
PUT    /api/admin/orders/:id        # 更新订单状态

GET    /api/admin/analytics/traffic     # 流量分析
GET    /api/admin/analytics/conversion  # 转化分析
GET    /api/admin/analytics/sales       # 销售分析
```

**公开 API（面向独立站）**：
```
GET    /api/products                # 获取产品（公开）
GET    /api/products/:slug          # 获取产品详情
GET    /api/cases                   # 获取案例（公开）
GET    /api/articles                # 获取文章（公开）
POST   /api/inquiries               # 提交询盘（公开）
```
